; /meuatalho {
;  /say linha1
;  /say linha2
;  /say linha3
; }
;
; ou usar a barra '|', por exemplo:
; /meuatalho /say linha1 | /say linha2 | /say linha3 | /say linha4 (e assim sucessivamente)

;
; *** Os textos acima estão como comentários devido ao ';' no início de cada linha, se você vai
; utilizá-los, cole-os abaixo da próxima linha e retire o ';' (veja os exemplos)
